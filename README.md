# Ether Registration Form

A responsive and interactive registration form with modern design and validation.

## Features
- ✅ Ether Name validation
- ✅ Email validation
- ✅ Password validation with show/hide toggle
- ✅ Confirm password matching
- ✅ Home page features selection
- ✅ Form validation with error messages
- ✅ Success message display
- ✅ LocalStorage for data persistence
- ✅ Responsive design
- ✅ Modern UI with gradients

## How to Use
1. Open `index.html` in any web browser
2. Fill in the registration form
3. Toggle password visibility with the eye icon
4. Select home page features
5. Click "Register" to submit
6. View success message and console logs

## Files
- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - Form validation and interactivity

## Technologies
- HTML5
- CSS3 (Flexbox, Grid, Gradients)
- JavaScript (ES6)
- Font Awesome icons

## Validation Rules
- Ether Name: Minimum 3 characters
- Email: Valid format required
- Password: Minimum 6 characters
- Confirm Password: Must match password
